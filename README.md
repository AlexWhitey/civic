### Civic
---

This is the core project for Hack Oregon's Civic Platform.

 Built with:
- **React** - view engine / server side rendering
- **React-Router** - client side routing
- **Express** - server
- **Redux** - state manager
- **Mocha/Enzyme** - testing
- **Webpack** - bundler
- **Yarn** - dependency manager
- **Docker** - containerization
